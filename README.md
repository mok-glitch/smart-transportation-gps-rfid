# Smart Transportation: GPS Tracking System (GPS + RFID)

This project introduces a GPS-RFID hybrid bus tracking system to ensure reliable location monitoring in urban areas.  
- **Technologies used:** Arduino UNO, GPS Module (GY-NEO6MV1), RFID, LCD Display  
- **Features:** Real-time GPS tracking with RFID checkpoints as backup.
